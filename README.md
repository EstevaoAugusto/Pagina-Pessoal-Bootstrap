# Pagina-Pessoal-Bootstrap

Este é um projeto da disciplina 'Programação WEB' do curso de Ciência da Computação na UFLA (Universidade Federal de Lavras). Nele foram criado duas paginas pessoais a fim de demonstrar as diferenças entre dois frameworks CSS, sendo eles: Bootstrap e Tailwind. Abaixo estão os links para cada uma das versões.

- Link do Bootstrap: https://estevaoaugusto.github.io/Pagina-Pessoal-Bootstrap/Bootstrap/
- Link do Tailwind: https://estevaoaugusto.github.io/Pagina-Pessoal-Bootstrap/Tailwind/
